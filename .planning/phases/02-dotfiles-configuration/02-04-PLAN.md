**Part A: Create `home/dot_tmux.conf.tmpl`**

Rewrite `.tmux.conf` (94 lines) as a chezmoi Go template. Key changes from the legacy config:

1. **TPM plugins section** — keep all existing plugins: tpm, tmux-sensible, tmux-yank, extrakto, tmux-net-speed, tmux-sidebar, tmux-prefix-highlight.

2. **Prefix and keybindings** — keep exactly as-is: prefix C-a (unbind C-b), splits i/o, vim nav h/j/k/l, mouse on, history 50000.

3. **REMOVE hardcoded powerline paths** — the existing config has:
   ```
   source-file "/opt/homebrew/lib/python3.9/site-packages/powerline/bindings/tmux/powerline.conf"
   ```
   This is macOS-only and version-pinned to Python 3.9. Instead, use a conditional:
   ```
   {{ if eq .chezmoi.os "darwin" }}
   # Powerline: brew install powerline-status
   if-shell "test -f /opt/homebrew/lib/python3.*/site-packages/powerline/bindings/tmux/powerline.conf" \
     "source-file /opt/homebrew/lib/python3.*/site-packages/powerline/bindings/tmux/powerline.conf"
   {{ else }}
   # Powerline: pip install powerline-status
   if-shell "test -f /usr/share/powerline/bindings/tmux/powerline.conf" \
     "source-file /usr/share/powerline/bindings/tmux/powerline.conf"
   {{ end }}
   ```
   Wait — tmux `source-file` doesn't support globs. Better approach: use `if-shell` with a find/test command, or simply skip powerline and rely on tmux-prefix-highlight + TPM status plugins. The existing powerline setup is fragile.

   **Decision: Remove powerline source-file entirely.** The tmux-prefix-highlight plugin + tmux-sensible already provide a functional status bar. The custom "Quang" status bar text and battery icons can remain as plain tmux status-right/status-left config.

4. **REMOVE custom script references** — the existing config references `$HOME/dotfiles/scripts/wan_ip.sh` and similar. These are fragile. Remove them from the status bar.

5. **Keep** — the basic status bar settings (colors, window-status-format) but clean up to use standard tmux settings without external script dependencies.

6. **TPM init at bottom** — `run -b '~/.tmux/plugins/tpm/tpm'`

7. **256 color and terminal settings** — keep `set -g default-terminal "tmux-256color"` and `set-option -ga terminal-overrides ",xterm-256color:Tc"`.

**Part B: Update `home/.chezmoiexternal.toml`**

Add TPM as a chezmoi external (similar to how OMZ is already managed):

```toml
    refreshPeriod = "168h"
```

This ensures `chezmoi apply` downloads TPM automatically, matching the pattern used for OMZ and P10k.
