Create `home/run_once_03_macos-wm-setup.sh.tmpl` — a chezmoi run_once_ script (macOS-only, template-gated) that performs one-time window manager setup. Per MACOS-04, the script MUST warn about SIP but NEVER exit non-zero due to SIP status.

Script structure (per D-convention from run_onchange_02):

```bash
#!/bin/bash
# chezmoi:template:left-delimiter="{{" right-delimiter="}}"
# macOS Window Manager Setup — Sketchybar + Yabai
# Runs once after chezmoi apply. Safe to re-run (idempotent).

{{ if ne .chezmoi.os "darwin" -}}
exit 0
{{ end -}}

set -euo pipefail

printf '\n══════════════════════════════════════════════\n'
printf '  macOS Window Manager Setup\n'
printf '══════════════════════════════════════════════\n\n'
```

**Section 1 — SIP Detection (MACOS-04):**
- Run `csrutil status` and capture output
- If output contains "enabled" → print a clearly-formatted WARNING block:
  - Explain that Yabai window management features require SIP disabled
  - Explain that Sketchybar works without SIP but some features may be limited
  - Point user to `https://github.com/koekeishiya/yabai/wiki/Disabling-System-Integrity-Protection`
  - State: "Continuing setup — configs will be applied but some features won't work until SIP is disabled and you reboot."
- If SIP is disabled → print "✓ SIP is disabled — full Yabai/Sketchybar support available"
- Do NOT exit non-zero in either case

**Section 2 — Install SbarLua:**
- Check if SbarLua is already installed: test if `/usr/local/lib/lua/5.4/sketchybar.so` exists (or similar path)
- If not installed:
  ```bash
  echo "Installing SbarLua..."
  (git clone https://github.com/FelixKratz/SbarLua.git /tmp/SbarLua && cd /tmp/SbarLua/ && make install && rm -rf /tmp/SbarLua/)
  ```
- If already installed: print "✓ SbarLua already installed"

**Section 3 — Build Sketchybar helper binaries:**
- Check if `~/.config/sketchybar/helpers` exists (it will after chezmoi copies configs in Plan 03-02)
- If exists: `cd "$HOME/.config/sketchybar/helpers" && make`
- If not: print "⚠ Sketchybar config not yet applied — run chezmoi apply first, then re-run this script"
- Note: This step may fail if configs haven't been applied yet (plan 03-02). That's OK — the helpers can be built manually later. Wrap in `|| true` to prevent script failure.

**Section 4 — Download Sketchybar app font:**
- Check if `$HOME/Library/Fonts/sketchybar-app-font.ttf` exists
- If not: `curl -fsSL https://github.com/kvndrsslr/sketchybar-app-font/releases/download/v2.0.5/sketchybar-app-font.ttf -o "$HOME/Library/Fonts/sketchybar-app-font.ttf"`
- If exists: print "✓ Sketchybar app font already installed"

**End:** Print summary line: "macOS WM setup complete."

Important: Use `set -euo pipefail` at top but wrap potentially-failing commands (SbarLua, helpers make, font download) with `|| true` or proper conditionals so the script ALWAYS exits 0.
