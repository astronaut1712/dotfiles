### Task 2.2: Create home/.chezmoi.toml.tmpl

<read_first>
- .planning/research/STACK.md (chezmoi.toml.tmpl pattern, promptOnce)
- .planning/research/ARCHITECTURE.md (chezmoi init flow)
</read_first>

<action>
Create `home/.chezmoi.toml.tmpl` with the following content:

```toml
{{- $email := promptStringOnce . "email" "Email address" -}}
{{- $name  := promptStringOnce . "name"  "Full name" -}}
{{- $op_available := false -}}
{{- if lookPath "op" -}}
  {{- $op_available = true -}}
{{- end -}}

[data]
    email         = {{ $email | quote }}
    name          = {{ $name | quote }}
    op_available  = {{ $op_available }}

[onepassword]
    command = "op"

[git]
    autoCommit = false
    autoPush   = false
```

This file is a template (`.tmpl`). On first `chezmoi init`, chezmoi will prompt once for email and full name, then write the static TOML to `~/.config/chezmoi/chezmoi.toml`. Values persist across re-runs (`promptStringOnce`).
</action>

<acceptance_criteria>
- `home/.chezmoi.toml.tmpl` exists
- File contains `promptStringOnce . "email"`
- File contains `promptStringOnce . "name"`
- File contains `lookPath "op"` (1Password availability check)
- File contains `[onepassword]` section
- `chezmoi execute-template < home/.chezmoi.toml.tmpl 2>&1 || true` — file is parsable template syntax (no bash errors on read)
</acceptance_criteria>

Create `home/.chezmoiexternal.toml` with the following content:

```toml
    url     = "https://github.com/ohmyzsh/ohmyzsh/archive/master.tar.gz"

    stripComponents = 1
    refreshPeriod = "168h"  # weekly

    type    = "archive"
    url     = "https://github.com/romkatv/powerlevel10k/archive/master.tar.gz"
    exact   = true
    stripComponents = 1
    refreshPeriod = "168h"

[".oh-my-zsh/custom/plugins/zsh-autosuggestions"]
    type    = "archive"
    url     = "https://github.com/zsh-users/zsh-autosuggestions/archive/master.tar.gz"
    exact   = true
    stripComponents = 1
    refreshPeriod = "168h"
```
</action>

<acceptance_criteria>
